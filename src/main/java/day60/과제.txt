[ 웹 과제6 : 3개월차 최종과제 ]
    1. 주제 : 요식업 웨이팅(대기 명단) 서비스 구축
    2. 개발 환경 요구사항
        1] 인텔리제이 에서 새로운 gradle 프로젝트 생성하여 진행합니다.
        2] Spring web MVC2 3tire 패턴 사용 하시오.
        3] HTML은 Thymeleaf 템플릿을 사용하시오.
    3. 서비스 요구사항
        1. '대기 명단 등록' 페이지 와 '대기명단 목록' 페이지 로 이동할 수 있게 헤더에 각 페이지 이동 버튼 만들어주세요.
        2. ['대기 명단 등록'] 웹에서 고객들에게 전화번호와 인원수를 입력받아 대기 번호를 주는 서비스 만들어주세요.
        3. ['대기 명단 목록'] 웹에서 관리자가 모든 대기 명단 목록을 볼 수 있도록 만들어주세요.

    * 추가 요구사항[ 시간이 남으면 ]
        1. ['대기 명단 등록'] 웹에서 현재 대기 명단 팀수 를 출력해주세요.
        2. ['대기 명단 목록'] 웹에서 모든 대기 명단 목록에서 특정한 대기 명단 개별 삭제를 할 수 있도록 삭제버튼을 만들어주세요.
        3. ['마이페이지'] 마이페이지를 추가하여 내 전화번호를 입력하면 현재 예상 시간을 알려주세요.
            -> 만약에 대기 명단에 없는 전화번호이면 '존재하지 않는 번호입니다'를 출력 해 주세요.
            -> 예상 시간은 대기 명단 1개 마다 5분씩 부여하여 만일 3개의 대기 명단이 존재하면 '대기 예상 시간 : 15분'을 출력해주세요.
        * CSS 선택.
